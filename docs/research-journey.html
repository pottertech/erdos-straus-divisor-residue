Research Journey: A Divisor-Residue Criterion for the Erdős–Straus Conjecture

This document narrates the research process from initial idea through eight manuscript revisions, conducted on July 18, 2026.

Phase 1: The A-Parameter Framework

The starting point was Mballa's (2026) parametric framework: write x = (n+A)/4 with A ≡ 3 (mod 4), reducing 4/n = 1/x + 1/y + 1/z to finding a divisor P of (nx)² with a specific residue mod A. The key insight was defining the bounded divisor-residue set D_A((nx)²) — the set of achievable residues mod A from actual divisors of (nx)², with exponents bounded by 2v_p(nx).

This is NOT a subgroup — it's a bounded subset. The distinction matters.

Phase 2: The Divisor-Residue Criterion (Theorem 1)

The exact characterization: A works iff T = −n²·4⁻¹ mod A lies in D_A. Verified with zero mismatches across 6,666 test cases (n ≤ 100,000).

Phase 3: Reduction to Quadratic Non-Residues (Theorem 3)

For prime A, the condition reduces to: some prime factor of nx is a QNR mod A. This connects the Erdős–Straus problem to quadratic reciprocity and the Burgess bound — classical analytic number theory.

Phase 4: The n ≡ 5 (mod 8) Case (Theorem 2)

Direct proof: when n ≡ 5 (mod 8), the prime 2 divides m₃ = (n+3)/4, and 2 ≡ −1 (mod 3), so P = 2 is a working divisor for A = 3. No polynomial identity needed.

Phase 5: Computational Verification

Exhaustive search confirmed: A ≤ 31 covers all n ≤ 100,000 (6,666 cases), and A ≤ 99 covers all but one of 666,666 cases up to 10,000,000. The Burgess bound gives A = O(n^{0.152}) conditionally.

Phase 6: The Bounded Divisor-Residue Lemma

The key technical lemma: −1 ∈ H(A) ⟹ −1 ∈ D_A. Proven in general for two cases:

  • h = 2 (5,533 cases): Single QNR prime with exponent 1
  • Order-2 QNR (445 cases): p¹ = g^{h/2} = −1 directly
  • Kneser condition (929 cases): Sumset covers Z/m when Σ(2eᵢ+1) ≥ m+k−1

Remaining 495 cases (7.6%) verified computationally with honest disclosure.

Phase 7: Reviewer Feedback (8 Revisions)

The manuscript went through eight revisions based on reviewer feedback:

  • v3 → v4: Fixed D_A vs H(A) — D_A ≠ H(A) in general (6 counterexamples)
  • v4 → v5: Removed "even cardinality" argument (D_A is not a subgroup). Downgraded lemma to "partially proven." Fixed Kneser application with correct stabilizer analysis.
  • v5 → v6: Fixed D_A closure issue — D_A is not closed under multiplication. Introduced shifted divisor-residue set D_A^{(nm)}.
  • v6 → v7: Made Theorem 2 independent of Theorem 3. Softened abstract Burgess claim.
  • v7 → v8: Labeled corollary as conditional. Softened evidence bullet.

  • Theorem 2: n ≡ 5 (mod 8) → A = 3 (direct proof)
