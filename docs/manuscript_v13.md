## 1. Introduction

The Erdős–Straus conjecture states that for every integer n ≥ 2, there exist positive integers x, y, z such that:

$$\frac{4}{n} = \frac{1}{x} + \frac{1}{y} + \frac{1}{z}$$

Despite work since 1950 (Erdős, Mordell, Vaughan, Webb, Elsholtz–Tao, Bright–Loughran, Mballa, Salez), the conjecture remains open. Mordell (1967) showed that polynomial identities providing solutions for n ≡ r (mod m) exist only when r is a quadratic non-residue mod m. Since 1 is a quadratic residue modulo every m, no polynomial identity can cover n ≡ 1 (mod m) — making n ≡ 1 (mod 12) the resistant case.

We present:
- A **divisor-residue criterion** reducing the conjecture to a bounded divisor search
- **Six modular identities** proven in Lean 4 (covering all n ≢ 1 mod 12, plus n ≡ 1 mod 12 with 5 | n)
- **Theorem 2** (n ≡ 5 mod 8, n prime) proven in Lean 4 via `linear_combination + Int.ofNat_inj`
- A **coverage theorem** proving every n ≥ 2 falls into one of 7 cases (6 proven + 1 open)
- **166,011 computational certificates** for all admissible primes n ≤ 10M, independently verified
- An explicit **axiomatic placeholder** for the remaining open case

## 2. Divisor-Residue Criterion

**Setting.** For n ≥ 2, we introduce a parameter A = 4x − n (with A ≡ 3 mod 4, gcd(n, A) = 1) and reduce the conjecture to a **bounded divisor-residue condition**: for each admissible n, there must exist bounded A and a divisor P of (n(n+A)/4)² satisfying P ≡ −n²·4⁻¹ (mod A) with positive y, z constructed as:

$$x = \frac{n + A}{4}, \quad y = \frac{P + N}{A}, \quad z = \frac{Q + N}{A}$$

where N = n(n+A)/4 and Q = N²/P.

**Status:** Proven on paper (algebraic derivation). Not yet formalized in Lean.

---

## 3. Centered Signed-Exponent Formulation

The divisor-residue set D_A(N²) can be characterized via **centered signed exponents**: residues generated by prime factors of nx with exponents bounded by 2v_p(nx), centered at ±e. This connects the divisor-residue condition to quadratic non-residue conditions via quadratic reciprocity.

**Status:** Proven on paper (group theory + character sums). Not yet formalized in Lean.

## 4. Proven Route Families

### 4.1 Modular Identities (Lean-verified)

The following six identities are proven in Lean 4 (`code/Identities.lean`) with `ring`:

| Residue class | Identity | Lean theorem |
|---|---|---|
| n ≡ 0 (mod 3) | 4/n = 1/n + 1/(2k) + 1/(2k), n = 3k | `mod_3_zero` |
| n ≡ 2 (mod 3) | 4/n = 1/n + 1/(k+1) + 1/(n(k+1)), n = 3k+2 | `mod_3_two` |
| n ≡ 4 (mod 12) | 4/n = 3/(3n/4) (symmetric) | `mod_12_four` |
| n ≡ 7 (mod 12) | 4/n = 4/(n+1) + 4/(n(n+1)) | `mod_12_seven` |
| n ≡ 10 (mod 12) | 4/n = 1/(n/2) + 2/n | `mod_12_ten` |
| n ≡ 1 (mod 12), 5 | n | 4/n = 5/(2n) + 1/n + 1/(2n), n = 60m+25 | `mod_12_one_mod_5_zero` |

### 4.2 Theorem 2: n ≡ 5 (mod 8), n prime (Lean-verified)

**Theorem.** *For every prime n ≥ 5 with n ≡ 5 (mod 8): there exist positive integers x, y, z with 4/n = 1/x + 1/y + 1/z.*

**Construction:** x = (n+3)/4, y = (2+nx)/3, z = nx(nx/2+1)/3, where nx = n·x.

**Proof:** The key additive relation 2w + 3x = 4x² (from 2w = nx, 4x = n+3) avoids `Nat.sub`. The polynomial identity is closed via `linear_combination` with exact coefficients in ℤ, transferred back via `Int.ofNat_inj`. (See `code/CenteredEquivalence.lean`.)

### 4.3 Coverage Theorem (Lean-verified)

**Theorem.** *Every n ≥ 2 falls into one of 7 residue classes: n ≡ 0 (mod 3), n ≡ 2 (mod 3), n ≡ 4 (mod 12), n ≡ 7 (mod 12), n ≡ 10 (mod 12), n ≡ 1 (mod 12) with 5 | n, or n ≡ 1 (mod 12) with 5 ∤ n.*

(See `code/MainTheorem.lean`, `residue_classification`.)

### 4.4 Main Theorem (Lean-verified)

**Theorem.** *For every n ≥ 2, excluding n ≡ 1 (mod 12), 5 ∤ n, n ≥ 13: IsErdosStraus n holds.*

(See `code/MainTheorem.lean`, `erdos_straus_proven_cases`.)

### 4.5 Theorem2 Integration (Lean-verified)

**Theorem.** *The broad unresolved class is n ≡ 1 (mod 12), 5 ∤ n, n ≥ 13. Within that class, the prime subfamily n ≡ 5 (mod 8) is proven by theorem2.*

(See `code/MainTheorem.lean`, `erdos_straus_except_residual_open_class`.)

## 6. Computational Certificate Verification

**Layer 4 certificates.** For every admissible prime n ≤ 10M (n ≡ 1 mod 12, 5 ∤ n, n ≥ 13), we generate an auditable certificate containing:

- n, A (first working A in search order [3, 7, 11, ..., 107])
- P (witness divisor of N² with P ≡ T mod A)
- Q (complementary divisor, Q = N²/P)
- T (target residue, T = -N mod A)
- N (= n(n+A)/4)
- x, y, z (explicit construction)
- verification: 4xyz = n(xy + xz + yz), P | N², P ≡ T mod A, x,y,z > 0

**Results:**
- 166,011 certificates generated, 0 failures
- 0 parse errors (regenerated clean JSONL)
- 100% pass rate under independent verifier (`analysis/layer4/verify_certificates.py`)
- Tested certificate bound C = 107 for admissible primes n ≤ 10M
- A-distribution: A=3 covers 78.4%, A=7 covers 17.2%, A=11–107 covers 4.4%
- Notable: n = 4,766,689 certified with A = 39; n = 8,803,369 certified with A = 107

**Certificates record exact witnesses, not route classifications.**

## 7. Remaining Open Class

The broad remaining open case is:

$$n \equiv 1 \pmod{12}, \quad 5 \nmid n, \quad n \geq 13$$

**Already proven within this class:**
- Prime n ≡ 5 (mod 8): proven by theorem2 (Lean-verified)

Note: n ≡ 0 (mod 5) is NOT part of the residual class — the class explicitly assumes 5 ∤ n. The case 5 | n is handled separately by `mod_12_one_mod_5_zero` and is outside the open class entirely.

**Still unresolved within the broad residual class** (all with n ≡ 1 mod 12, 5 ∤ n, n ≥ 13):
- Composite n ≡ 5 (mod 8) — theorem2 requires primality, so composites are not covered
- Composite n ≡ 1 (mod 8)
- Prime n ≡ 1 (mod 8)

**Evidence for the open case:**
- 166,011 computational certificates, all verified
- Mballa (2026) proved density-1 coverage for n ≡ 1 (mod 4)
- No closed-form parametric formula known
- The original formula (x = 3m+1 or 3m+2) breaks for large n (e.g., n = 7477)

The open case is declared as an `axiom` in Lean with a docstring explicitly marking it as "CONJECTURE (not proven)."

## 8. Analytic Route and Future Work

The Burgess-style estimate gives an analytic upper-bound route under stated hypotheses. Combining it with the covering-set framework suggests:

$$A = O(n^{1/(4\sqrt{e})+\varepsilon})$$

for the remaining prime cases. However, the manuscript does not claim this as a completed proof until the sieve and covering assumptions are fully formalized.

**Recommended next formalization targets:**
1. Formalize the divisor-residue criterion in Lean (connects certificates to theorems)
2. Formalize the centered signed-exponent equivalence in Lean
3. Produce certificate-checking theorem statements in Lean

**What this project is:**
A structured reduction plus formalized subtheorems plus an independently auditable finite certificate layer.

**What this project is not:**
A full proof of the Erdős–Straus conjecture.
